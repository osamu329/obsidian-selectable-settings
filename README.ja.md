# Selectable Settings Text

Obsidianの設定画面で、設定項目名や説明文をテキスト選択・コピーできるようにするCSSスニペットです。

## できること

デフォルトのObsidianでは、設定画面の項目名や説明文をドラッグ選択・コピーすることができません。このスニペットを適用すると、設定画面内のテキストを自由に選択してコピーできるようになります。

## インストール

1. `selectable-settings.css` をダウンロード
2. `<vault>/.obsidian/snippets/` に配置
3. **設定 → 外観 → CSSスニペット** を開き、**selectable-settings** を有効化
