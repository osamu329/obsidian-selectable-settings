# Selectable Settings Text

An Obsidian CSS snippet that makes setting names and descriptions selectable and copyable in the Settings panel.

## What this does

By default, Obsidian's Settings panel prevents you from selecting or copying the text of setting item names and descriptions. This snippet adds `user-select: text` to those elements, allowing you to drag-select and copy any text you see in the Settings panel.

## Installation

1. Download `selectable-settings.css`
2. Place it in `<vault>/.obsidian/snippets/`
3. Open **Settings → Appearance → CSS snippets** and enable **selectable-settings**
